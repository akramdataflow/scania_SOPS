
# قارئ مواصفات سكانيا (FPC) – Streamlit

تطبيق بسيط يقرأ ملف XML لشاحنة سكانيا ويستخرج رموز **FPC** (الاسم/القيمة) ويطابقها مع معانيها
من دليل PDF (أو ملف CSV بديل)، ثم يعرض المواصفات في جدول قابل للتنزيل.

## المتطلبات
- Python 3.9+
- انصب الحزم:
```bash
pip install -r requirements.txt
```

## التشغيل
```bash
streamlit run app.py
```

## الاستخدام
1. في الشريط الجانبي:
   - ارفع ملف الشاحنة بصيغة **XML**.
   - ارفع ملف الدليل بصيغة **PDF** (أو CSV يحتوي الأعمدة: `name_code,value_code,meaning`).
   - (اختياري) عدّل اسم وسم عناصر FPC إن كانت مختلفة عن `FPC` في XML.
2. سترى جدولًا يضم `name_code` و `value_code` و `meaning`.
3. نزّل النتائج كـ CSV أو Excel.
4. إن بقيت قيَم بلا معنى، استعمل المحرر اليدوي لإضافة الوصلات مؤقتًا، ثم نزّل النتائج.

> ملاحظة: استخراج الـ PDF يعتمد على هيكل الملف. إن لم ينجح، يرجى تصدير جدول القيم إلى CSV واستخدامه مباشرة.
